
PROJECT DESCRIPTION:
This project is a web-based SMS spam detection system developed using
HTML, CSS, and JavaScript. The user enters a text message into the
application, and the system analyzes the message using basic NLP
techniques to determine whether it is spam or not.
OBJECTIVES:
1. To design a simple web application for detecting spam messages.
2. To understand the basic concepts of Natural Language Processing (NLP).
3. To classify messages into Spam or Ham (Not Spam).
4. To demonstrate real-time text analysis using JavaScript.
5. To provide an easy and user-friendly interface for message checking.

TOOLS AND TECHNOLOGIES USED:
Frontend:
- HTML       : To create the structure of the web page.
- CSS        : To style and design the user interface.
- JavaScript : To implement spam detection logic.
Concepts:
- Natural Language Processing (NLP)
- Text preprocessing
- Keyword-based classification
Browser:
- Google Chrome (for execution and testing)
